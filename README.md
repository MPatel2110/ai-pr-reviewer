# AI PR Reviewer

An open-source GitHub Action that posts AI-generated code review comments on pull requests, with pluggable LLM backends.

**Status:** Under active development. Day 1 of 7-day build.

## Planned Features

- AI-powered inline review comments on pull requests
- Pluggable LLM provider system (Gemini implementation first)
- Structured JSON output for reliable comment generation
- Per-repo YAML configuration
- Context-aware reviews (pulls related project files)
- Cost-optimized with prompt caching
